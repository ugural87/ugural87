# Uğur Alkan

Quantitative modeller — thirteen years of mathematical modelling across domains, from
continuum mechanics and constitutive modelling to financial markets. Currently completing
an MSc in Data Analytics (Sabancı University, 2026).

I build models the same way regardless of domain: explicit assumptions, leakage-free
protocols, calibrated probabilities, and an honest statement of what the data can and
cannot support. The unifying thread is that **models are not the deliverable — decisions
are.**
